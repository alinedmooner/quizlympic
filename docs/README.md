# Quizlympic
 
## Rules

- Rooms can only be created by registered users.

- Before creating the room you can choose two modes:
  
    - Single room.
    - Multiple room.

- Once the room is created, the following options can be configured:

  - The game time can be 5, 10 or 15 minutes.
  - Initially the tests will be on specific topics and will have subcategories by difficulty.
